# BounceNet
Open Source Matchmaking Service for Multiplayer Games


# Local Build
setup local MongoDB
set MONGODB_URI environment variable (default is mongodb://127.0.0.1:27017)
set BOUNCENET_SECRET environment variable (no default)

# Deploy to Heroku
set MONGODB_URI environment variable (default is mongodb://127.0.0.1:27017)
set BOUNCENET_SECRET environment variable (no default)
